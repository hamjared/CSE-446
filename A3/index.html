The application I created for this assignment is a basic console app, that allows the user to login and send messages. It was originally planned to make use of the message sending service on the ASU service repository, but that service was not working as of the deadline for this project. So I just used the file store service in the ASU service repository instead. It functions well enought for the purposes of this project. I created the login/ Sign up services used by the application. Both of these functions make use of Code Activities.
  • CreateUser(string userName, string password)
    • Takes a user name and password, checks if the user name is in use and if not creates the user
  • login(string userName, string password)
    • Takes a user name and password and searches the users json file stored in app data for a matching user name, then checks if the passwords match. If they do the login is successful
